# 冒泡排序 BubbleSort
    冒泡排序是一种比较简单的 稳定排序 算法，效率不高，因此实际当中用到的机会并不多。但
    作为快速排序算法的基础，还是有必要了解一下。
    顾名思义，冒泡就是指大的数字（气泡）会优先从底部升到顶端（大的数浮力较大，上升）。
    具体实现的方式就是重复的 【交换排序】，通过两两比较不断地将大的数字右移。
